# 💳 Credit Card Fraud Risk Analysis Dashboard

## 📖 Project Overview

Credit card fraud is one of the biggest challenges faced by financial institutions. Detecting suspicious transactions quickly can reduce financial losses and improve customer trust.

This project analyzes credit card transaction data to identify fraud patterns, risk levels, high-risk regions, and transaction categories contributing most to fraudulent activity.

The dashboard was built using Power BI, where raw transaction data was cleaned, transformed, modeled, and analyzed using DAX measures to generate actionable business insights.

## 🎯 Business Objective

The goal of this project is to help fraud investigation teams answer critical business questions:

- Which fraud type causes the highest financial loss?
- Which states experience the most fraud cases?
- What percentage of transactions are high-risk or critical-risk?
- Which transaction categories require stronger monitoring?
- Are there specific months when fraud activity spikes?

---

## 🧹 Data Preparation

The dataset contained multiple real-world data quality challenges:

- Missing values
- Duplicate records
- Inconsistent fraud labels
- Incorrect data formats
- Unstructured categorical values
- Transaction amount anomalies

To ensure reliable analysis:

- Cleaned and standardized transaction records
- Removed duplicate entries
- Handled missing values
- Created fraud risk classifications
- Built relationships between tables
- Developed DAX measures for KPI calculations

## 🔍 Key Insights

### Fraud Type Analysis

Card Not Present transactions generated the highest fraud amount, indicating a major vulnerability in digital payment channels.

### Risk Distribution

A significant share of fraudulent transactions falls under Medium and High Risk categories, helping investigators prioritize cases effectively.

### Regional Analysis

Certain states consistently report higher fraud volumes, highlighting the need for region-specific fraud controls.

### Trend Analysis

Fraud activity increases during specific months, suggesting seasonal fraud behavior that requires proactive monitoring.
